# Type Alias: DDOpts

> **DDOpts** = `"enable"` \| `"disable"` \| `"destroy"` \| `"option"` \| `string` \| `any`

表示拖拽相关的操作选项类型。
可以是以下值：
- 'enable': 启用拖拽功能
- 'disable': 禁用拖拽功能
- 'destroy': 销毁拖拽实例
- 'option': 设置拖拽选项
- string: 其他字符串命令
- any: 任意其他值
